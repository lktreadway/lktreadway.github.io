Lottery Research Reports

Under construction.
My motto: Chaos does not exist.
There is only a level of complexity yet to be discovered and understood.

I find this to be true for most problem-solving tasks.
I love numbers and puzzles. During my career break, I created approximately twenty Java programs that produced lottery statistical reports based on the book Lottery Master Guide by Gail Howard. I used spreadsheets as the data input source and wrote the output to the console, transfering it into other spreadsheets or word documents to study.
I intend to either rewrite all these classes in Node.js, or simply add new input/output mechanisms for the existing Java code.

I find lottery numbers, or any set of data, fascinating to study.
Statistical biases and pattern recognition is a favorite hobby of mine.
In time, I plan to have a membership website that will provide a host of staticial reports for just about any lottery game in the U.S.

This site build is currently in process.
The build will use the following:
Client-side: HTML, CSS, JavaScript. Server-side: Node.js/Java and MongoDB/MySQL.
