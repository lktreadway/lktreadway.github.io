OCP Java SE 11 (Exam 1Z0-819) Certification

Expected Completion: First quarter 2024

This certification is a comprehensive test for the Java 11 JDK, statistically the most widely used version of Java to-date.

I bought hard copies of the following books:
OCP Java SE 11 Developer Study Guide
OCP Java SE 11 Developer Practice Tests

The study guide contains 1100 pages covering every topic relevant to the Java 11 language. I got as far as chapter 2 before I realized that reading it straight through would not work for the way my brain learns. I now use these texts as reference material instead.

I decided I would be better off taking a different approach to studying for this exam. Instead of waiting to use the OCP Practice Tests book, I decided to use it up front to discover where my weaknesses lie.

The last time I took the Java 5 exam there were only a few areas where I did poorly causing me to fail the exam by inches. Considering I didn't study for the exam before taking it and that these areas were not utilized in my day-to-day work, I did very well. However, that was then and this is now.

The JDK has grown exponentially. I will utilize study techniques that I know work best for me. To that end, after determining the areas of focus that are needed, I will seek out online learning materials to fill in the gaps in my knowledge.

Lottery Research Reports

Under construction.
My motto:
Chaos does not exist. There is only a level of complexity yet to be discovered and understood.

I find this to be true for most problem-solving tasks.

I love numbers and puzzles. During my first career break, I created approximately twenty Java programs that produced lottery statistical reports based on the book Lottery Master Guide by Gail Howard. I used spreadsheets as the data input source and wrote the output to the console, transfering it into other spreadsheets or Word documents to study.

I intend to either rewrite all these classes in Node.js, Python, or simply add new input/output mechanisms for my existing Java code.

I find lottery numbers, or any set of data, fascinating to study. Statistical biases and pattern recognition is a favorite hobby of mine.

In time, I plan to have a membership website that will provide a host of staticial reports for just about any lottery game in the U.S.

This site build is currently in the planning phase. The build will use the following: Client-side: HTML, CSS, JavaScript. Server-side: Node.js/Python/Java and MongoDB/MySQL/SQLite.
